读书方法：

第一遍：快速过一遍+示例代码。

第二遍：认真过一遍+习题。

第三遍：重点、难点、错题、思维导图、博客（总结）。



记录掌握程度：5个等级，用1-5表示。

- 等级1: 除了字认识，其他都不认识（0-3成）。
- 等级2: 理解5成左右（4-7成）。
- 等级3: 基本理解（8-10成，看到题目就有思路）
- 等级4: 理解并能实现。
- 等级5: 理解并能快速实现（20分钟50行）
