* 这是一个行缓冲和全缓冲的问题。
* 标准输出定向到终端时，是行缓冲，每次打印一行。
* 标准输出定向到文件时，是全缓冲，fork的时候，复制了缓冲区内容，故而有此结果（当缓冲区满或关闭文件的时候，冲洗缓冲区）
